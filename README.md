# Contact Form with JavaScript Validation

##  Project Overview
This project is a simple contact form built using **HTML**, **CSS**, and **JavaScript**.  
It validates user inputs on the **client side** before allowing submission.  
The form includes:
- Name
- Email
- Phone Number
- Subject
- Category dropdown
- Agreement checkbox
- Message textarea

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
